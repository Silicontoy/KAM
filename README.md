# KAM

KAM (Knowledge Application Methodology) is a four-layer decoupled methodology for transforming domain expertise into AI-ready capabilities.

## Architecture

```
Layer 1: Professional Knowledge  (Domain Expert)     → Initial Data + Label Definitions
Layer 2: Knowledge Processing    (Knowledge Engineer)→ Knowledge Blocks + Flow Plans
Layer 3: AI Tool Planning        (AI Architect)      → Technical Solution Document
Layer 4: Tool Development        (Developer)         → Executable AI Tool
```

## Core Concepts

- **Knowledge** = Specific Information + Data Flow + Control Flow
- **Control Flow**: Script → Model → Skill → Agent
- **Data Flow**: Training → Fine-tuning → RAG → Context Learning
